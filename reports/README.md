# 🛡️ Cyber Guard AI

Cyber Guard AI is an AI-powered cybersecurity platform designed to detect phishing emails, analyze suspicious URLs, and help users identify potential cyber threats in real time.

## 🚀 Features

- 📧 Phishing Email Detection
- 🔗 Malicious URL Analysis
- 🤖 AI-Powered Threat Assessment
- 📄 PDF Report Generation
- 📊 Threat Risk Scoring
- 🎨 User-Friendly Interface
- 🔒 Cybersecurity Awareness Support

## 🛠️ Technologies Used

- Python
- Flask
- HTML, CSS, JavaScript
- Machine Learning
- Groq API
- ReportLab
- Scikit-learn
- Pandas
